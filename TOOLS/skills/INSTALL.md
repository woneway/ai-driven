# Skills 安装指南

## 自动安装

在终端运行以下命令安装技能：

### Swift / iOS

```bash
# SwiftUI 专家技能
npx skills add avdlee/swiftui-agent-skill --skill swiftui-expert-skill -y -g

# iOS 设计技能  
npx skills add wshobson/agents --skill mobile-ios-design -y -g
```

### Python

```bash
# Python 测试技能
npx skills add wshobson/agents --skill python-testing-patterns -y -g

# Python 性能优化
npx skills add wshobson/agents --skill python-performance-optimization -y -g
```

### Java

```bash
# Spring 开发技能
npx skills add mindrally/skills --skill java-spring-development -y -g

# Java 架构师技能
npx skills add jeffallan/claude-skills --skill java-architect -y -g
```

### 测试

```bash
# 端到端测试
npx skills add wshobson/agents --skill e2e-testing-patterns -y -g

# Web 测试
npx skills add anthropics/skills --skill webapp-testing -y -g
```

## 手动安装步骤

1. 打开终端
2. 运行上述命令
3. 等待安装完成

## 验证安装

```bash
npx skills list
```
