# Java / Spring 开发技能

## 可用技能

### java-spring-development

- **描述**: Spring 开发最佳实践
- **功能**: REST API、Spring Security、Spring Data JPA、测试
- **安装**: `npx skills add mindrally/skills --skill java-spring-development`

### spring-framework

- **描述**: Spring Framework 技能
- **功能**: Spring Boot、Maven、JUnit
- **安装**: `npx skills add mindrally/skills --skill spring-framework`

### spring-boot-dependency-injection

- **描述**: Spring 依赖注入
- **功能**: 构造函数注入、Bean 管理
- **安装**: `npx skills add giuseppe-trisciuoglio/developer-kit --skill spring-boot-dependency-injection`

### java-spring-boot

- **描述**: Spring Boot 技能
- **功能**: Spring Boot 3.x、生产级应用
- **安装**: `npx skills add pluginagentmarketplace/custom-plugin-java --skill java-spring-boot`

### java-architect

- **描述**: Java 架构师技能
- **功能**: 微服务、Clean Architecture、SOLID
- **安装**: `npx skills add jeffallan/claude-skills --skill java-architect`

## 推荐安装

```bash
npx skills add mindrally/skills --skill java-spring-development
npx skills add jeffallan/claude-skills --skill java-architect
```
