# Python 开发技能

## 可用技能

### python-testing-patterns

- **描述**: Python 测试技能
- **功能**: pytest、mocking、TDD
- **安装**: `npx skills add wshobson/agents --skill python-testing-patterns`

### python-performance-optimization

- **描述**: Python 性能优化
- **功能**: 性能分析、优化技巧
- **安装**: `npx skills add wshobson/agents --skill python-performance-optimization`

### async-python-patterns

- **描述**: Python 异步编程
- **功能**: asyncio、异步模式
- **安装**: `npx skills add wshobson/agents --skill async-python-patterns`

### python-packaging

- **描述**: Python 包发布
- **功能**: pip、pypi 发布
- **安装**: `npx skills add wshobson/agents --skill python-packaging`

## 推荐安装

```bash
npx skills add wshobson/agents --skill python-testing-patterns
npx skills add wshobson/agents --skill python-performance-optimization
```
