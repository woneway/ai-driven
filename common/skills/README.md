# Skills 技能库

ai-driven 使用的技能库。

## 目录结构

```
skills/
├── AVAILABLE.md           # 所有可用技能列表
├── swift/               # Swift / iOS 技能
├── python/              # Python 技能
├── java/                # Java / Spring 技能
├── testing/             # 测试技能
├── superpowers/         # 核心开发技能
└── diagram/             # 图表技能
```

## 安装技能

```bash
npx skills add <owner/repo> --skill <skill-name>
```

详细列表见 `AVAILABLE.md`

## 按语言分类

- **Swift / iOS**: `swift/` 目录
- **Python**: `python/` 目录
- **Java**: `java/` 目录
- **测试**: `testing/` 目录
