# Swift / iOS 开发技能

## 可用技能

### swiftui-expert-skill

- **描述**: SwiftUI 专家技能
- **功能**: 状态管理、现代 API、性能优化、iOS 26+ Liquid Glass
- **安装**: `npx skills add avdlee/swiftui-agent-skill --skill swiftui-expert-skill`
- **仓库**: https://github.com/AvdLee/SwiftUI-Agent-Skill

### mobile-ios-design

- **描述**: iOS 设计技能
- **功能**: UI/UX 最佳实践
- **安装**: `npx skills add wshobson/agents --skill mobile-ios-design`

### xcode-build-skill

- **描述**: Xcode 构建技能
- **功能**: xcodebuild、simctl 使用
- **安装**: `npx skills add pzep1/xcode-build-skill --skill xcode-build`

## 推荐安装

```bash
npx skills add avdlee/swiftui-agent-skill --skill swiftui-expert-skill
npx skills add wshobson/agents --skill mobile-ios-design
```
