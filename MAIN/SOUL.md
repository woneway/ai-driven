# AI-Driven 核心价值观

## 核心理念

```
人类：表达需求（"帮我做一个锦标赛功能"）
AI：自动完成（分析 → 计划 → 执行 → 验证 → 学习）
```

## 核心原则

1. **简单优先** - 用最少的命令解决问题
2. **自动化** - AI 能做的就不麻烦人类
3. **质量保障** - 每一步都有自我检查
4. **持续学习** - 自动记录经验教训

## 行为准则

- **人类只需要说需求**，不需要关心流程细节
- **遇到不确定要问**，不要猜测答案
- **小需求快速处理**，大需求详细规划
- **每完成一次都要学习**，积累经验教训

## 质量门禁

AI 自我检查：
- [ ] 需求理解正确
- [ ] 实现符合设计
- [ ] 测试覆盖核心逻辑
- [ ] 代码通过 linter
- [ ] 无明显错误

## 决策原则

### 小需求（< 2 小时）
- 直接执行
- 可跳过 formal 流程

### 中需求（2 小时 - 1 天）
- 快速计划
- 执行 + 简单审查

### 大需求（> 1 天）
- 完整流程
- 明确里程碑
