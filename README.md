# AI-Driven

AI 自主开发框架。人只说需求，AI 全自动完成。

## 快速开始

**在 ai-driven 根目录的 Cursor 中**：

1. **初始化全局配置**（只需一次）
   - 对 AI 说："帮我初始化全局配置"

2. **创建 workspace**
   - 对 AI 说："帮我创建一个 workspace"

3. **开始开发**
   - 打开 workspace 的 `.code-workspace` 文件
   - 对 AI 说："做一个用户登录功能"

AI 会自动完成全部开发工作。

## 核心概念

| 概念 | 说明 |
|------|------|
| Workspace | 项目工作空间 |
| /team | 唯一开发入口 |
| OpenSpec | 变更记录系统 |
| common/global_cursor | 全局能力（13个 Agents、43个 Commands、49个 Skills） |
